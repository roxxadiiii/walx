## 📖 About

This repository contains the wallpaper collection used by **WalX**.

The wallpapers are organized to be consumed directly by the Hugo-based WalX gallery and are automatically displayed through Git submodules and GitHub Pages deployments.

## ✨ Features

* 📁 Organized wallpaper collection
* 🖼️ High-resolution images
* 🚀 Optimized for Hugo GalleryDeluxe
* 🔄 Designed to be used as a Git submodule
* 🌐 Automatically published through the WalX website

## 📂 Repository Structure

```text
content/images/
├── Nature/
├── Anime/
├── Minimal/
├── Abstract/
├── Dark/
├── Landscape/
└── ...
```

## 🔗 Usage

This repository is intended to be used as a Git submodule:

```bash
git submodule add https://github.com/roxxadiiii/walx-images.git content/images
```

Then initialize:

```bash
git submodule update --init --recursive
```
